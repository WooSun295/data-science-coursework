# Data Science Coursework Repository
This repository contains graduate-level coursework projects focused on statistical analysis, machine learning, dimensionality reduction, clustering, forecasting, and scientific data analysis using Python and Jupyter Notebooks. The projects emphasize both theoretical understanding and practical implementation across real-world datasets in healthcare, bioinformatics, computer vision, and oceanographic forecasting.

## Projects Included
### 1. Statistical Analysis and Data Preprocessing on the Penguins Dataset
Comprehensive exploratory data analysis, covariance/correlation analysis, scaling methods, and univariate/bivariate statistical analysis using the Palmer Penguins dataset.
### 2. Feature Selection and Dimensionality Reduction for Gene Expression Data
Analysis of high-dimensional cancer gene expression data using feature selection methods, PCA, and t-SNE for visualization and biomarker discovery.
### 3. Predictive Modeling for Heart Failure Patients
Regression and classification modeling for heart failure clinical data using Linear Regression, Ridge/Lasso Regression, Logistic Regression, PCA, and feature selection techniques.
### 4. Emotion Image Clustering with PCA, t-SNE, and Feature Selection
Clustering grayscale facial-expression images using K-Means, Gaussian Mixture Models, and Agglomerative Clustering with dimensionality reduction and feature selection workflows.
### 5. Forecasting Weekly Sea Surface Height Using EOF and Time-Series Models
Spatiotemporal oceanographic data analysis using EOF decomposition, ARMA/ARIMA/Kalman filtering, and reconstruction of future sea surface height fields.
